### JavaScript

### Requirements

[Node](https://formulae.brew.sh/formula/node).

Check we have it installed with:

```bash
$ node --version
v14.7.0
```

### Quick Start

```bash
# jump in to this directory
cd javascript

# install the dependencies
npm install

# run the tests
npm test

# watch the tests
npm run test:watch
```
