# Haskell

### Requirements

[Haskell stack](https://docs.haskellstack.org/en/stable/README/#how-to-install).

Check we have it installed with:

```bash
$ stack --version
Version 2.3.1 x86_64 hpack-0.33.0
```

### Quick Start

```bash
# jump in to this directory
cd haskell

# run the tests 
stack test

# run the repl
stack ghci
```
